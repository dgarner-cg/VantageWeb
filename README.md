# Vantage 🔭



**The professional high-ground for community management.**



Vantage is a sophisticated Discord based community management assistant with an easy to use web-integrated platform designed to help community managers cultivate healthy, safe, and flourishing digital spaces. We bridge the gap between complex server logistics and intuitive management, allowing you to focus on the things you're good at, that being primarily spending you creative energy on helping your community to grow without losing its soul. 
Vantage is the operating system that allows that to happen. We handle all of the management, moderation and automation infrastructure so you can enjoy what you're building.



---



## 🚀 The Vision



We believe that as digital communities scale, communication shouldn't just get louder and more disorganized, it should get better and conversations should get more robust and filled with creativity. Vantage is built on three core pillars:



* **Safety First:** Vantage offers advanced moderation features that work towards enforcing safety without penalizing participants. We want to act as a shield, not a hammer.

* **Professional Oversight:** Vantage offers a dedicated web dashboard that really is the heart and soul of our platform, providing community managers a "vantage point" over server defense and protection, settings and configurations, insights, server health, moderation, automations and more; all without having to spend hours on configuring settings or having a technical background.

* **Flourishing Environments:** Features designed to reduce friction and encourage meaningful, positive member interaction. 



## ✨ Key Features



| Feature | Description |
